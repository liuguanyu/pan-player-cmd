# 流光效果实现说明

## 效果展示

程序启动时会显示一个彩色流光文字动画，文字逐字显示，每个字符使用不同的颜色，呈现出流光溢彩的效果。

## 实现原理

### 1. 核心组件

流光效果主要依赖以下组件：

- **定时器（Timer）**: 使用 `tea.Tick` 创建定时器，每隔一定时间触发一次更新
- **状态管理**: 记录当前显示到第几个字符
- **颜色渐变**: 为每个字符分配不同的颜色

### 2. 关键代码结构

#### 消息类型定义

```go
// TickMsg 定时器消息
type TickMsg struct{}

// SplashAnimationDoneMsg 流光动画完成消息
type SplashAnimationDoneMsg struct{}

// LoadingAnimationMsg 加载动画消息
type LoadingAnimationMsg struct{}
```

#### 状态字段

```go
type App struct {
    // 流光效果状态
    splashText      string  // 要显示的文字
    splashIndex     int     // 当前显示到第几个字符
    splashAnimating bool    // 是否正在动画中

    // 加载动画状态
    loadingFiles bool
    loadingDots  int  // 加载动画的点的数量
}
```

### 3. 实现步骤

#### 步骤 1: 初始化动画

```go
func (a *App) startSplashAnimation() tea.Cmd {
    a.splashAnimating = true
    a.splashIndex = 0
    return a.tick()  // 启动定时器
}
```

#### 步骤 2: 定时器触发

```go
func (a *App) tick() tea.Cmd {
    // 每100毫秒触发一次
    return tea.Tick(100*time.Millisecond, func(t time.Time) tea.Msg {
        return TickMsg{}
    })
}
```

#### 步骤 3: 处理定时器消息

```go
case TickMsg:
    if a.currentView == ViewSplash && a.splashAnimating {
        if a.splashIndex < len(a.splashText) {
            a.splashIndex++  // 增加显示的字符数
            return a, a.tick()  // 继续下一次定时
        } else {
            // 动画完成
            a.splashAnimating = false
            return a, a.waitForSplash()
        }
    }
```

#### 步骤 4: 渲染流光效果

```go
func (a *App) renderSplashContent() string {
    displayText := a.splashText[:a.splashIndex]

    // 定义颜色序列
    colors := []string{
        "#FF6B6B", "#FF8E53", "#FFC857", "#C9E4CA",
        "#87CEEB", "#B4A7D6", "#FF69B4", "#7D56F4",
    }

    var styledText strings.Builder
    for i, ch := range displayText {
        // 为每个字符选择颜色（循环使用颜色数组）
        color := colors[i%len(colors)]
        charStyle := lipgloss.NewStyle().
            Foreground(lipgloss.Color(color)).
            Bold(true)
        styledText.WriteString(charStyle.Render(string(ch)))
    }

    return styledText.String()
}
```

### 4. 加载动画效果

在文件浏览器加载文件时，也有类似的流光效果：

if a.loadingFiles {

## 关键技术点

### 1. Bubbletea 框架的定时器

```go
tea.Tick(duration, func(t time.Time) tea.Msg {
    return YourMsg{}
})
```

定时器会在指定时间后发送一个消息，触发状态更新。

### 2. Lipgloss 样式

```go
lipgloss.NewStyle().
    Foreground(lipgloss.Color("#FF6B6B")).
    Bold(true).
    Render(text)
```

使用 Lipgloss 可以轻松地为终端文本添加颜色和样式。

### 3. 渐变色彩方案

选择和谐的颜色序列，循环使用以创建流动的视觉效果：

```go
colors := []string{
    "#FF6B6B",  // 红色
    "#FF8E53",  // 橙色
    "#FFC857",  // 黄色
    "#C9E4CA",  // 绿色
    "#87CEEB",  // 天蓝色
    "#B4A7D6",  // 淡紫色
    "#FF69B4",  // 粉色
    "#7D56F4",  // 紫色
}
```

## 应用场景

这种流光效果可以用于：

1. **启动画面**: 程序启动时的欢迎信息
2. **加载状态**: 文件加载、网络请求等长时间操作的提示
3. **重要提示**: 需要用户注意的关键信息
4. **品牌展示**: 程序名称或 logo 的展示

## 性能考虑

- 定时间隔不要太短（建议 100ms 以上），避免过度刷新
- 动画要能够被中断或跳过，不要阻塞用户操作
- 在文本较长时，可以考虑加快速度或减少颜色变化

## 扩展思路

可以进一步扩展的功能：

1. **波浪效果**: 字符逐个放大缩小
2. **移动渐变**: 颜色从左到右流动
3. **闪烁效果**: 文字透明度变化
4. **打字机音效**: 配合声音增强体验
5. **自定义速度**: 允许用户调整动画速度
